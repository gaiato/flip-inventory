# Flip Inventory Photos

Public image CDN for lot-flip inventory listings. Photos served as raw URLs for embedding in Notion, marketplace listings, and other platforms.

## Structure

```
{item-slug}/
  image0.jpeg
  image1.jpeg
  ...
```

## Usage

Raw URLs follow this pattern:
```
https://raw.githubusercontent.com/gaiato/flip-inventory/main/{item-slug}/image{N}.jpeg
```

Example: `https://raw.githubusercontent.com/gaiato/flip-inventory/main/elitebook-840-g8-32gb/image0.jpeg`
